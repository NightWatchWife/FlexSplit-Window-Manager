Privacy Policy

最終更新日 / Last updated: 2026-06-27

FlexSplit Window Manager は、いかなるユーザーデータも収集・保存・送信・共有しません。
FlexSplit Window Manager does not collect, store, transmit, or share any user data.

1. 収集する情報 / Information we collect

本拡張機能は、個人情報、閲覧履歴、ページ内容、認証情報、位置情報、利用状況の分析（アナリティクス）を含め、いかなるユーザーデータも一切収集しません。データを外部サーバーへ送信することもありません。すべての処理はお使いの端末内で完結します。

The extension collects no user data of any kind — no personal information, browsing history, page content, credentials, location, or analytics. Nothing is sent to any external server. All processing happens entirely on your device.

2. ウィンドウ整列のために参照する情報 / What the extension reads

ウィンドウの移動・リサイズを行うため、本拡張機能は次の配置情報のみをお使いの端末上で参照します。これらは読み取られるだけで、保存も送信もされません。

  • ウィンドウの位置・サイズ (window position and size)
  • タブの並び順・所属ウィンドウ・固定状態 (tab order, window membership, pinned state)

タブの URL・タイトル・ページ内容にはアクセスしません。It never accesses tab URLs, titles, or page content.

3. 権限 / Permissions

本拡張機能はいかなる権限も要求しません（permissions の宣言なし）。使用する Chrome API はすべて権限不要のものです。

The extension declares no permissions. All Chrome APIs it uses require none.

4. 第三者への提供 / Third parties

データを収集しないため、第三者へ販売・転送・共有するものはありません。広告・トラッキング・解析サービスは一切利用していません。

Because no data is collected, there is nothing to sell, transfer, or share with third parties. No advertising, tracking, or analytics services are used.

5. 本ポリシーの変更 / Changes to this policy

本ポリシーを更新する場合は、本ページと更新日を改訂します。

If this policy changes, this page and its "last updated" date will be revised.
