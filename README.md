# FlexSplit Window Manager

Chrome 拡張機能（Manifest V3）。現在のタブを、選んだ分割数（2〜6）で画面に均等整列させます。

## 機能

- 拡張アイコンのポップアップから **2/3/4分割・MAXボタン** または **ドロップダウン（2〜6分割・MAX）** でワンクリック整列。
- **MAX**：対象範囲で開いているタブ数だけ分割（上限10分割）。
- **まとめる**：分割の逆操作。対象範囲の複数ウィンドウのタブを1つのウィンドウに集約して最大化。
- **対象範囲**を選択可能：
  - アクティブなモニター（既定）— ポップアップを開いたモニター上のウィンドウのタブを対象
  - アクティブなブラウザ — 今フォーカスしている1ウィンドウのタブのみ
  - すべてのウィンドウ — 全モニター・全ウィンドウのタブ
- 先頭から N 個のタブを取り出し、`width = availWidth / N` で横並びに配置。1番目は元ウィンドウを左端へ、2番目以降は新規ウィンドウに切り離して配置。
- 端数は最後の列が吸収し、Windows の不可視枠ぶんを補正して隙間を低減。
- タブ数が分割数より少ない場合は警告を表示してスキップ。
- **日英対応（i18n）**：ブラウザのUI言語に応じて自動で日本語/英語を表示（既定は英語）。

## インストール（開発者モード）

1. `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をオン
3. 「パッケージ化されていない拡張機能を読み込む」→ このフォルダを選択

## 隙間の微調整

`background.js` の定数 `BORDER`（既定 8px）で、ウィンドウ間・周囲の隙間を調整できます。
表示スケール（100% / 125% / 150% など）により最適値が変わります。

- 隙間が残る → 値を大きく
- 重なる／中身が切れる → 値を小さく
- 補正なし → `0`

## ファイル構成

| ファイル | 役割 |
|---|---|
| `manifest.json` | Manifest V3 設定（popup / service worker） |
| `popup.html` | 分割数・対象範囲を選ぶ UI |
| `popup.js` | UI 操作を検知し、画面サイズと設定を background へ送信 |
| `background.js` | タブの取得・分割・配置を実行（Service Worker） |
| `_locales/en/messages.json` | 英語の文言 |
| `_locales/ja/messages.json` | 日本語の文言 |
| `icons/icon*.png` | 拡張アイコン（16/32/48/128px） |
