## English

### Name
FlexSplit Window Manager

### Summary (short description, ≤132 chars)
Tile your browser windows evenly in one click — 2 to 6 panes or MAX, per-monitor scope, and one-click merge back.

### Category
Productivity

### Detailed description

Stop dragging and resizing windows by hand. FlexSplit arranges your browser
windows into a clean, even grid across the screen — in a single click.

Features
• One-click split — choose 2, 3, 4, 5, 6, or MAX. Each option is a live preview
  of the layout it creates.
• MAX — tile as many windows as you have open tabs (up to 10).
• Merge back — gather scattered windows into one maximized window with a single
  button. The perfect undo for a split.
• Pick your scope — tile only the active monitor, only the active browser
  window, or every window across all monitors.
• Edge-to-edge fit — the last column absorbs leftover pixels and an adjustable
  border compensation keeps gaps minimal on Windows.
• English & Japanese — the interface follows your browser language automatically.

How to use
1. Click the FlexSplit toolbar icon.
2. Pick a scope (Active monitor / Active browser / All windows).
3. Click a layout (2–6 or MAX) to tile instantly — or "Merge into one" to gather.

Privacy
FlexSplit does not collect, store, or transmit any data. It only uses the
`tabs` and `windows` permissions to read tab/window positions and move and resize
your windows on your own machine.

## 日本語

### 名前
FlexSplit Window Manager

### サマリー（短い説明・132文字以内）
ブラウザのウィンドウをワンクリックで画面に均等整列。2〜6分割・MAX、対象範囲の選択、ワンボタンで1つに集約も。

### カテゴリ
生産性向上（Productivity）

### 詳細な説明

ウィンドウを手作業でドラッグ＆リサイズするのはもう終わり。FlexSplit は、開いている
ブラウザのウィンドウを、画面いっぱいのきれいな均等グリッドにワンクリックで整列します。

主な機能
・ワンクリック分割 — 2 / 3 / 4 / 5 / 6 / MAX から選ぶだけ。各ボタンは「実際に並ぶ
  レイアウト」のプレビューになっています。
・MAX — 開いているタブの数だけ分割（上限10分割）。
・まとめる — 散らばったウィンドウを1つのウィンドウに集約して最大化。分割のちょうど逆操作。
・対象範囲を選択 — アクティブなモニターだけ / アクティブなブラウザだけ / すべての
  モニターの全ウィンドウ、から選べます。
・端まできっちり — 端数ピクセルは最後の列が吸収。Windows の不可視枠ぶんを補正し、
  ウィンドウ間の隙間を最小化します。
・日英対応 — ブラウザの言語に合わせて自動で表示が切り替わります。

使い方
1. ツールバーの FlexSplit アイコンをクリック。
2. 対象範囲を選択（アクティブなモニター / アクティブなブラウザ / すべてのウィンドウ）。
3. レイアウト（2〜6 または MAX）をクリックして即整列。集約は「まとめる」ボタン。

プライバシー
FlexSplit はいかなるデータも収集・保存・送信しません。`tabs` と `windows` の権限は、
タブ／ウィンドウの位置を読み取り、お使いの端末上でウィンドウを移動・リサイズするため
だけに使用します。
